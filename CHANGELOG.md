# Changelog

All notable changes to **SVG Peek** will be documented in this file.

## [0.0.1] — Initial release

- Hover preview for inline `<svg>` blocks in `.svg`, `.xml`, `.html`, `.js`, `.ts`, `.jsx`, `.tsx`, `.vue`, and `.md`.
- **Open in Tab** link in every hover — shows the SVG in a side panel with a transparency checkerboard.
- Folding regions for every `<svg>` block, plus `Fold All` / `Unfold All` commands.
- Document formatter for `.svg` / `.xml` and a command to format embedded SVG blocks in any other file.
- Optional gutter thumbnails, toggled via command palette or the `svgPeek.gutter.enabled` setting.
